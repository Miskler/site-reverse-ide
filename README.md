# Canvas Links

Простой редактор блоков и связей на Flask + чистом JavaScript.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

После запуска открой `http://127.0.0.1:5000`.

## Что умеет

- Добавлять блоки на холст.
- Перетаскивать блоки мышью.
- Создавать направленные связи между блоками.
- Редактировать название, описание и цвет выбранного блока.
- Удалять блоки и связи.
- Сохранять состояние на сервере и в `localStorage` как запасной вариант.
